# PlayENG 3000

Tanulj angolul mondatokban, ne szavakban. Az Oxford 3000 leggyakrabban használt angol szó – azonnal használható mondatokkal.

## Az első lépés a legnehezebb – kezdd el ma

### Futtatás Docker-rel

```bash
docker-compose up --build -d
```

A weboldal elérhető: http://localhost:3000

### PlayENG Módszer

A PlayENG módszer a feleségem, Greta által kifejlesztett 10 éves tapasztalaton alapuló nyelvtanulási módszer. A tanulók nem szavakat tanulnak – azonnal mondatokat alkotnak.

### Tech Stack

- **Frontend**: React.js + Apple-inspired design
- **Backend**: Python Flask API
- **Deploy**: Docker Compose
